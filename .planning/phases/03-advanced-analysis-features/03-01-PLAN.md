  <action>
    Create `backend/app/services/advanced_analysis_service.py` with class `AdvancedAnalysisService`:

    1. **Constructor**: `__init__(self, session: AsyncSession)` - stores session.

    2. **_load_daily_data(ts_code)** - Copy the pattern from vpa_service.py: SELECT DailyBar WHERE ts_code=ts_code ORDER BY trade_date, convert to DataFrame with columns [trade_date, open, high, low, close, vol].

    3. **detect_pivots(ts_code)** - Core pivot detection per Chapter 07 "isolated pivot" algorithm:
       - Iterate bars from index 1 to len-1 (need left and right neighbors).
       - Isolated high pivot: bar[i].high > bar[i-1].high AND bar[i].high > bar[i+1].high AND bar[i].low > bar[i-1].low AND bar[i].low > bar[i+1].low
       - Isolated low pivot: bar[i].low < bar[i-1].low AND bar[i].low < bar[i+1].low AND bar[i].high < bar[i-1].high AND bar[i].high < bar[i+1].high
       - Return list of {trade_date, type: "high"|"low", price: float (high for high pivot, low for low pivot), bar_index: int}

    4. **detect_support_resistance(ts_code)** - Per D-01/D-02:
       - Get pivots from detect_pivots.
       - Group pivot prices within 2% tolerance: for each pivot, check if any existing level has price within abs(pivot_price - level_price) / level_price < 0.02. If yes, add to that group; otherwise create new level.
       - For each group: price = mean of group prices, strength = count of pivots in group.
       - Classify: levels above current close price = "resistance", levels below = "support", levels within 0.5% of close = "both".
       - Return {levels: [{price, type, strength, trade_dates: [str]}]}

    5. **detect_trend_lines(ts_code)** - Per D-04/D-05, Chapter 08 dynamic trend lines:
       - Get pivots from detect_pivots.
       - Separate into high_pivots (sorted by bar_index) and low_pivots (sorted by bar_index).
       - For high pivots: connect consecutive pairs where price trend is clear. Line from pivot[i] to pivot[i+1] if they are within 60 bars apart. Extend the line to current date using the slope (price_diff / bar_diff).
       - For low pivots: same logic.
       - Classify trend: if slope > 0, type="up"; if slope < 0, type="down"; if slope ~= 0, type="horizontal".
       - Return {lines: [{start_date, end_date, start_price, end_price, type, pivot_type: "high"|"low"}]}

    6. **detect_market_cycle(ts_code)** - Per D-06/D-07, Chapter 05 four phases:
       - Load data into DataFrame, add derived columns: ma20 = close.rolling(20).mean(), vol_ma20 = vol.rolling(20).mean(), atr14 = (high-low).rolling(14).mean().
       - Use a sliding window approach (window_size=20) to classify each segment:
         - **accumulation**: vol < vol_ma20 * 0.8 AND atr14 < atr14_mean * 0.7 AND close near ma20 (within 1 std dev) AND recent low_pivots present
         - **markup**: close > ma20 AND close making higher highs (close > close.shift(5)) AND vol > vol_ma20 on up days
         - **distribution**: vol > vol_ma20 * 1.2 AND atr14 < atr14_mean * 0.8 AND close near recent high AND high_pivots forming ceiling
         - **markdown**: close < ma20 AND close making lower lows AND vol > vol_ma20 on down days
         - Default: use the previous phase if no clear pattern
       - Merge consecutive same-phase segments into continuous phase ranges.
       - Return {phases: [{phase, start_date, end_date, description}]}

    7. **compute_vap(ts_code, start_date=None, end_date=None)** - Per D-09/D-10, Chapter 09:
       - Load data. If start_date/end_date provided, filter to that range.
       - Calculate price range: min(low) to max(high) in visible range.
       - Bin into 30 price levels: bin_size = (max_price - min_price) / 30.
       - For each bar, distribute its volume across price bins that overlap with [low, high] range. Use linear interpolation: vol_per_bin = vol * (bin_overlap / bar_range).
       - Track up_volume (close >= open) and down_volume (close < open) separately per bin.
       - Return {vap: [{price_level, volume, up_volume, down_volume}], price_range: {min, max}, total_volume}

    8. **aggregate_kline(ts_code, timeframe="weekly")** - Per D-12/D-13/D-14:
       - Load daily data.
       - If timeframe="weekly": group by ISO year + week number (Monday start). For each group: trade_date=first date, open=first open, high=max(high), low=min(low), close=last close, vol=sum(vol).
       - If timeframe="monthly": group by YYYYMM. Same aggregation.
       - Return {timeframe, data: [{trade_date, open, high, low, close, vol}]}

    9. **detect_divergence(ts_code)** - Per VPA-04:
       - Load data into DataFrame. Add columns: close_20_high = close.rolling(20).max(), vol_ma20 = vol.rolling(20).mean().
       - Find bars where close == close_20_high (new 20-day high) but vol < vol_ma20 * 0.8 (volume below average).
       - Return {divergences: [{trade_date, price, volume, vol_ma20, description}]}

    - Chapter 07 isolated pivot: 3-bar pattern where middle bar is "isolated" - its range is completely above (high pivot) or below (low pivot) both neighbors

    - Chapter 09 VAP: distribute each bar's volume across overlapping price bins, separate up/down volume
