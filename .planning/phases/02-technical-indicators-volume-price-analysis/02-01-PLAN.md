    **3. Create backend/app/services/indicator_service.py**:

    This service computes indicators using pandas + ta library for MACD/RSI/BOLL, and a custom KDJ implementation per D-14.

    Key design per D-13, D-16, D-17:
    - `compute_indicators(session, ts_code, indicator_name, params)` -- main entry point
    - Loads DailyBar data from DB as pandas DataFrame
    - Checks indicator_values cache first (by ts_code + indicator_name + params_hash)
    - If cache miss: compute, store to indicator_values table, return results
    - If cache hit: return cached results

    Indicator computation details:
    - **MACD**: Use `ta.trend.MACD` with params `fastperiod` (default 12), `slowperiod` (default 26), `signalperiod` (default 9). Returns DataFrame with columns: macd_line, signal_line, histogram.
    - **RSI**: Use `ta.momentum.RSIIndicator` with param `window` (default 14). Returns Series rsi.
    - **KDJ**: CUSTOM implementation per D-14. Use Stochastic Oscillator formula:
      - K = 2/3 * prev_K + 1/3 * (C - Ln) / (Hn - Ln) * 100  (n=9 default)
      - D = 2/3 * prev_D + 1/3 * K  (smooth period=3 default)
      - J = 3 * K - 2 * D
      - Initialize K=50, D=50 for first values
      - Clamp J to [0, 100]
    - **BOLL**: Use `ta.volatility.BollingerBands` with params `window` (default 20), `window_dev` (default 2). Returns upper, middle, lower bands.

    params_hash generation: `hashlib.md5(json.dumps(params, sort_keys=True).encode()).hexdigest()[:16]`

    value_json per indicator per date:
    - MACD: `{"macd": 0.12, "signal": 0.10, "histogram": 0.02}`
    - RSI: `{"rsi": 65.3}`
    - KDJ: `{"k": 72.1, "d": 68.5, "j": 79.3}`
    - BOLL: `{"upper": 25.5, "middle": 23.0, "lower": 20.5}`

    Return format from compute_indicators:
    ```python
    {
        "indicator": "macd",
        "params": {"fastperiod": 12, "slowperiod": 26, "signalperiod": 9},
        "params_hash": "abc123...",
        "data": [
            {"trade_date": "20240101", "macd": 0.12, "signal": 0.10, "histogram": 0.02},
            ...
        ]
    }
    ```

    **1. Create backend/app/services/vpa_service.py**:

    VPAService class with two main methods:

    `detect_volume_price_signals(session, ts_code)` -- returns volume-price analysis signals per D-06, D-07:

    Signal detection logic (reference: user notes chapter 4):
    - **Confirmation signals** (D-06):
      - `volume_up_rise`: close > open (up day) AND vol > avg_vol_5 * 1.2 (volume significantly above 5-day average)
      - `volume_down_fall`: close < open (down day) AND vol < avg_vol_5 * 0.8 (volume below 5-day average)
    - **Anomaly signals** (D-07):
      - `long_candle_low_volume`: body_ratio > 0.6 (large body relative to range) AND vol < avg_vol_5 * 0.7
      - `short_candle_high_volume`: body_ratio < 0.3 (small body) AND vol > avg_vol_5 * 1.5
      - `rising_volume_decline`: check last 3 bars -- all close > prev close but vol is decreasing each bar

    Where body = abs(close - open), range = high - low, body_ratio = body / range (guard range > 0)
    avg_vol_5 = rolling 5-day average volume

    Return format:
    ```python
    {
        "signals": [
            {
                "trade_date": "20240115",
                "signal_type": "volume_up_rise",  # or other type
                "direction": "up",  # "up" or "down"
                "description": "放量上涨"  # Chinese description
            },
            ...
        ]
    }
    ```

    `detect_kline_patterns(session, ts_code)` -- returns K-line pattern detections per D-08:

    Pattern detection logic (reference: user notes chapter 6):
    - **hammer** (锤头线): lower_shadow >= 2 * body AND upper_shadow <= body AND body > 0
      - where body = abs(close - open), lower_shadow = min(open, close) - low, upper_shadow = high - max(open, close)
    - **shooting_star** (射击十字星): upper_shadow >= 2 * body AND lower_shadow <= body AND body > 0
    - **doji** (十字星): body / range < 0.1 where range = high - low > 0
    - **hanging_man** (吊人线): same shape as hammer (lower_shadow >= 2 * body AND upper_shadow <= body) BUT only when it appears in an uptrend context (close > open for the bar, and close is above 5-day MA)

    Return format:
    ```python
    {
        "patterns": [
            {
                "trade_date": "20240115",
                "pattern_type": "hammer",
                "name": "锤头线",
                "description": "看涨信号：下影线长，实体小，出现在下跌趋势中"
            },
            ...
        ]
    }
    ```
