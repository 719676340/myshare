# 量价交易学习平台 — 项目规划

## 技术栈

| 层级 | 技术 |
|------|------|
| 前端 | Vue + JS + ECharts |
| 后端 | Python FastAPI |
| 数据库 | SQLite |
| 数据源 | tushare（A 股日 K 数据） |

## 项目结构

三个独立页面，顶部导航切换：

### 模块一：策略分析

通过 tushare 获取 A 股日 K 数据，结合量价分析策略，在 K 线图上直观展示分析结果。

**功能列表：**

- K 线图 + 成交量柱状图展示
- 量价确认/异常标记（放量上涨、缩量下跌、量价背离等）
- K 线形态识别（锤头线、射击十字星、十字星、吊人线等）
- 技术指标计算展示（MACD、RSI、KDJ、BOLL 等），计算后存入数据库
- 支撑/阻力位自动识别（孤立支点检测）
- 动态趋势线绘制
- 市场循环阶段标注（吸筹 → 上涨 → 派发 → 下跌）
- VAP 价量分布图
- 多时间跨度联动分析

**分阶段迭代：**

| 阶段 | 内容 |
|------|------|
| 第一阶段（核心） | K 线图 + 成交量 + 量价确认/异常标记 + 基础 K 线形态识别 |
| 第二阶段（进阶） | 支撑/阻力位 + 动态趋势线 + 市场循环阶段标注 |
| 第三阶段（高级） | VAP 价量分布图 + 多时间跨度联动 + 量烛图/等量图 |

**量价分析核心逻辑（基于笔记）：**

- 确认：长阳线 + 高成交量、短阳线 + 低成交量、趋势中量逐根放大
- 异常：长阳线 + 低成交量（陷阱）、短阳线 + 高成交量（走弱）、上涨中量递减（买压枯竭）
- K 线七元素分析：实体大小、上下影线长度反映市场情绪
- 市场循环：快速下跌 → 吸筹 → 买入高峰 → 供给测试 → 缓慢上涨 → 加速上涨 → 派筹 → 抛售高峰 → 需求测试 → 快速下跌
- 突破确认：放量突破为真，缩量突破为伪

### 模块二：交易练习

使用历史数据，用户不知道未来走势，根据已展示的信息进行逐日交易决策。

**功能列表：**

- 选定股票后，逐日推进，不可回退，模拟真实交易
- 操作：买入 / 卖出 + 仓位比例选择
- 初始资金可自定，默认 100 万
- 只做多（不做空）
- 模拟交易费用：佣金万 2.5 + 印花税千 1（卖出时收取）
- 最终统计收益率

**交易练习流程：**

1. 用户选择一只股票 + 设定练习的时间范围
2. 展示起始日之前的 K 线数据
3. 用户根据已有信息决定：买入/卖出/不操作 + 仓位比例
4. 确认操作后，揭示下一天的 K 线数据
5. 重复步骤 3-4，直到练习时间范围结束
6. 展示最终收益、交易记录、每笔盈亏

### 模块三：自动回测

预设量价策略模板，用户调整参数后自动回测历史收益率。

**功能列表：**

- 预设策略模板 + 参数调节（如成交量阈值、回看天数、MA 周期等）
- 回测指标输出：
  - 总收益率
  - 年化收益率
  - 最大回撤
  - 交易次数
  - 胜率
  - 盈亏比
  - 夏普比率
  - 平均持仓天数
- 可视化：资金曲线图（策略净值 vs 基准净值）
- 每笔交易买卖点标记在 K 线图上

## 数据管理

- tushare 拉取 A 股日 K 数据（OHLCV），存入 SQLite
- 技术指标（MACD、RSI、KDJ、BOLL 等）基于日 K 数据计算，结果存入数据库
- 支持预定义指标 + 参数调节
- 支持基于基础字段（开盘价、收盘价、成交量等）的四则运算组合新指标

## UI / 交互设计

| 项目 | 方案 |
|------|------|
| 整体风格 | TradingView 风格，默认深色主题 |
| 股票选择 | 搜索框输入名称/代码，模糊匹配，下拉选择 |
| K 线图交互 | 鼠标悬浮显示 OHLCV、十字光标、缩放、拖拽平移、区间框选 |
| 标注功能 | 后续迭代：手动画趋势线、标记支撑/阻力位 |

## 其他约定

- 无用户系统，单人使用
- 仅支持 A 股市场
- 仅使用日 K 线数据
- A 股颜色惯例：红涨绿跌
