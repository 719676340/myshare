# 第三章：合理的价格

> "熊市不言底，牛市不言顶。" —— 佚名

## 一、核心观点

> 只看价格 = 线上拍卖，看不到竞拍者数量和热情程度。没有成交量的价格分析只是故事的一半。

作者批评了**价格行为交易（PAT）**——仅用价格数据、几乎不用成交量的方法。如果利弗莫尔和威科夫知道有人只用价格不用成交量，"一定会目瞪口呆"。

## 二、场内交易员的困境（精彩类比）

过去伦敦交易所的场内交易员能**直接感受到**成交量：
- 看见买卖指令流
- 感受贪婪与恐惧的气氛
- "看见"大买家入场

转入电子化交易后，大多数场内交易员**转型失败**——因为他们失去了"看见成交量"的能力。这从反面证明了：**没有成交量，价格分析就不完整**。

## 三、电子化交易对价格四要素的影响

价格四要素：**开盘价、最高价、最低价、收盘价**

| 过去 | 现在 |
|------|------|
| 交易所开/收市才有交易，开盘价/收盘价极其重要 | 24小时电子交易，开盘价/收盘价重要性下降 |
| 跳空缺口常见且重要 | 跳空缺口罕见，多数会回补 |
| 开盘"跳空"能感受市场强烈情绪 | 夜盘期货提前消化了情绪 |

> 1992年芝加哥商品交易所引入Globex全球电子交易系统，几乎改变了所有市场的交易方式。

> 这使得**成交量在电子化时代尤为重要**——因为价格本身的信号变弱了。

## 四、K线图的七元素与市场情绪

作者只使用**K线图（蜡烛图）**，一根K线包含七个元素：
- 开盘价、收盘价、最高价、最低价
- 上影线、下影线、实体

### 各元素含义

| 元素 | 含义 |
|------|------|
| **高实体** | 强烈的市场情绪（看涨或看跌） |
| **低实体** | 较弱的市场情绪，买卖双方都不强 |
| **长下影线** | 卖方先占主导 → 买方反攻收复失地 → **收盘时情绪看涨** |
| **长上影线** | 买方先占主导 → 卖方反攻压回 → **收盘时情绪看跌** |
| **无影线** | 从始至终单边情绪，极度强烈 |

### 影线详解（重点）

**下影线 = 看涨信号**
1. 开盘后卖方主导，价格下跌
2. 跌到某价位，买方觉得"够便宜了"，入场
3. 买方逐渐夺回控制权，价格回升至开盘价
4. 关键洞察：情绪发生了**完全反转**（从看跌到看涨）

**上影线 = 看跌信号**
1. 开盘后买方主导，价格上涨
2. 涨到某价位，买方力竭，卖方入场
3. 卖方逐渐夺回控制权，价格回落至开盘价
4. 关键洞察：情绪发生了**完全反转**（从看涨到看跌）

## 五、两个重要概念区分

| 概念 | 含义 |
|------|------|
| **VPA（量价分析）** | 每根K线形成**后**，成交量与价格的关系——"大图" |
| **VAP（价量分布）** | 每根K线形成**过程中**，各价位的成交量特征——"细节" |

两者互为补充，相互验证。

## 六、本章结论

K线的影线能看出买卖行为，但**不能揭示**：
- 价格行为的强度
- 价格行为是否真实有效

> 这就是为什么价格行为分析只是技术分析的一部分，**成交量是另一部分**。
