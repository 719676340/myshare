# 第八章：动态趋势及趋势线

> "损失不是运气不佳造成的，而是错误的分析造成的。" —— 大卫·艾因霍恩

## 一、作者对传统趋势分析的批评

- 等到3个高位高点/低位低点画趋势线时，趋势已快结束
- 在历史图上画线毫无技术含量
- 等你进场时，局内人已准备离场
- "让趋势成为你的朋友" — 毫无意义的陈词滥调

## 二、道氏理论回顾

### 趋势级别

| 趋势级别 | 道氏时代 | 现代日内交易 |
|----------|---------|-------------|
| 主要趋势 | 2~3个月 | 2~3周 |
| 次要趋势 | 2~3周 | 2~3天 |
| 小趋势 | 2~3天 | 2~3小时 |

### 趋势三阶段（与威科夫对应）
1. **吸筹阶段** — 局内人/"聪明资本"买入
2. **趋势跟踪阶段** — 公众参与
3. **派筹阶段** — "聪明资本"清仓离场

## 三、核心方法：动态趋势线

**关键思想**：趋势线不是事后画的，而是在价格运动过程中**动态构建**的。

### 步骤详解（以上升趋势为例）

1. **放量突破**震荡区间 → 建仓进场
2. 等待第一个**高位支点**（3柱体逆转）
3. 价格回调，等待**低位支点**确认
4. 低位支点比上一个高 → 继续持有
5. 高位支点比上一个高 → 上升趋势确认
6. 用支点动态画出趋势的上下边界

### 关键原则

| 要素 | 含义 |
|------|------|
| 高位支点**越来越高** | 上升趋势健康 |
| 低位支点**越来越高** | 上升趋势健康 |
| 高位支点与前一持平或更低 | **警告信号**，可能进入新震荡 |
| 低位支点与前一持平或更低 | **警告信号**，趋势可能反转 |

## 四、支点 = 趋势的"路标"

- 支点在价格运动中动态形成
- 标示趋势的边界，像路边的路灯
- 每个支点告诉你：你在趋势中的什么位置
- 配合量价分析，给你持有头寸的信心

## 五、量价分析在趋势中的两个应用角度

| 角度 | 用途 |
|------|------|
| 震荡期间 | 确认是否为重大反转（买入高峰/抛售高峰） |
| 突破时刻 | 确认趋势的大小和真实性 |

## 六、趋势中成交量的判断

| 情况 | 判断 |
|------|------|
| 上涨 + 放量 | 趋势健康，继续持有 |
| 上涨 + 平量/缩量 | **缺乏动力**，趋势可能不持续 |
| 回调 + 缩量 | 正常回踩，继续持有 |
| 支点未出现 + 缩量 | 市场可能较弱，进入更高水平盘整 |

## 七、反趋势交易

- 与主要趋势方向相反的操作
- 两个特征：风险更大（逆浪游泳）、持仓时间更短
- 可以操作，但必须清醒意识到你在做什么

## 八、核心理念

> **价格震荡是趋势的孵化器，远比已经形成的趋势重要。**

- 震荡不是令人沮丧的 — 而是孕育下一个趋势的阶段
- 在趋势开始时进场（震荡突破的瞬间），而不是趋势末端
- 支撑/阻力是一切趋势的源头
- 静态趋势线没有错，但动态趋势线让你在趋势形成时就进场
