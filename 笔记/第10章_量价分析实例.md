# 第十章：量价分析实例

> "交易中有两个关键因素：正确的行动和足够的耐心。" —— 查尔斯·布兰德斯

## 一、本章概述

作者跨越多个市场和平台的真实案例演示：

| 市场 | 例子 | 时间跨度 |
|------|------|---------|
| 美国股票 | 霍尼韦尔（HON）、杜克能源（DUK） | 日线 |
| 商品ETF | 白银ETF、黄金ETF（GLD） | 5分钟、日线 |
| 外汇 | 澳元/美元 | 15分钟、周线 |
| 期货 | YM E-mini道指、ES E-mini标普、咖啡期货 | 5分钟、10分钟、跳动点图 |
| 指数 | 道琼斯工业平均指数 | 周线 |

## 二、经典底部反转序列

```
放量下跌 → 低实体+高量（异常=放量止跌）→ 锤头线+高量 → 
缩量回调（洗掉不坚定者）→ 放量突破 → 确认上涨
```

## 三、经典顶部反转序列

```
缩量上涨（警告）→ 高量低实体（异常=放量止涨）→ 射击十字星+高量 → 
连续射击十字星+成交量递增 → 确认下跌
```

## 四、经典多头陷阱

```
前一晚：射击十字星+极高量（庄家离场）
次日开盘：大阳线+低成交量（陷阱！）
散户追入 → 即随下跌，散户被套
```

## 五、各案例核心教训

### 霍尼韦尔（HON）— 耐心等待确认
- 放量止跌信号出现后，不是立刻反转
- 缩量横盘中低实体阴线+高量 = 卖压被吸收
- 放量跳空突破 = 真突破

### 杜克能源（DUK）— 油轮需要时间停下
- 多次锤头线和放量止跌信号出现但市场仍未反转
- 耐心等缩量测试确认卖压被完全吸收后再进场
- 跳空高开+低量+前方阻力位 = 警惕多头陷阱

### 白银ETF — 短线交易的量价逻辑
- 止损线可设在锤头线影线下方
- 谨慎交易者 vs 激进交易者的不同选择
- 所有趋势和交易机会都是相对的

### 黄金ETF — 逐根K线分析
- 下跌中放量止跌信号反复出现但不被确认 = 继续下跌
- 天量抛售（600万 vs 平均50万）= 恐慌性抛售
- 上涨中量价背离 = 极弱信号

### 澳元/美元 — 趋势中持仓的信心
- 下跌趋势中反弹+缩量 = 暂时反弹，继续持有空头
- 没有放量止跌信号 = 买方未真正进入
- 周线图上的抛售高峰可以持续18个月

### 期货开盘 — 主力的经典陷阱
- 开盘时段是主力测试和设陷阱的高发时段
- 大阳线+低成交量 = 多头陷阱
- 大阳线+高成交量 = 真实运动

## 六、成交量分析的灵活性

- 不需要每根K线的成交量都递增 — 只需整体高于平均
- 季节性因素、节假日会影响成交量
- 极端成交量会扭曲对其他量柱的判断
- 关键是寻找**异常**，而非追求完美递增/递减

## 七、跳动点图 vs 时间图

| 特点 | 时间图 | 跳动点图 |
|------|--------|---------|
| 柱体形成 | 固定时间间隔 | 固定交易笔数 |
| 市场活跃时 | 15分钟一根K线 | 几秒一根K线 |
| 市场清淡时 | 15分钟一根K线 | 几分钟一根K线 |
| 优势 | 直观、普遍 | 直观感受市场活跃度 |

> 许多专业交易员只用跳动点图，因为它让你"看到市场的内部运动"。

## 八、长期投资（道琼斯周线图）

- 吸筹阶段可持续4~5个月
- 吸筹结束后稳步上涨，成交量平稳
- 直到出现抛售高峰信号前，趋势不会反转
- 成交量永不可能被掩盖
