# 第七章：支撑位和阻力位

> "金钱与市场永远不会失去记忆，但人们却会忘记。" —— 肯尼斯·费雪

## 一、核心定位

- 量价分析 = 地基
- 支撑/阻力 = 墙壁、天花板、地板
- 为量价分析搭建框架，帮助判断市场所处阶段

## 二、市场三种运行方式

- 上涨、下跌、横盘震荡
- 横盘震荡占约 **70%** 时间，明显趋势只占 **30%**

### 横盘震荡的三个原因
1. 等待重大新闻公布
2. 局内人在吸筹/派筹
3. 价格运行到过去的套牢区间，交易者平仓

## 三、支撑位与阻力的本质

> 支撑位和阻力位并没有什么神奇魔力，它们只是反映了特定价格区间上人们**极端的恐惧和贪婪**。

| 概念 | 比喻 | 本质 |
|------|------|------|
| **支撑位** | 地板 / 弹簧垫 | 价格下跌到这里被"托住"反弹 |
| **阻力位** | 天花板 / 无形障碍 | 价格上涨到这里被"压回" |

### 阻力位形成原因
在高位买入的弱势交易者，等价格回到成本时急于卖出 → 形成抛压

### 支撑位形成原因
在低位买入的强势交易者，在回调到成本附近时愿意持有/加仓 → 形成支撑

## 四、孤立支点 — 识别震荡区间的关键

### 孤立高位支点（三柱体逆转）
中间柱体的最高点 > 两侧柱体最高点，最低点 > 两侧柱体最低点

### 孤立低位支点
中间柱体的最低点 < 两侧柱体最低点，最高点 < 两侧柱体最高点

### 判断流程
1. 出现孤立高位支点 → 第一个信号
2. 出现孤立低位支点 → 震荡区间确立
3. 高位支点 = 天花板，低位支点 = 地板

## 五、突破的确认

| 突破类型 | 价格特征 | 成交量特征 | 回踩特征 |
|----------|---------|-----------|---------|
| **多头突破** | 收盘价明确突破天花板 | **放量**（必须！） | 回踩天花板伴随**低量** |
| **空头突破** | 收盘价明确突破地板 | **放量**（必须！） | 回踩地板伴随**低量** |
| **伪突破** | 略微突破后回落 | **无量/缩量** | 局内人设的陷阱！ |

> 推高市场就像把人从沼泽中拉出来 — 需要很大力气，必须体现在成交量上。

## 六、三条原则

| # | 原则 | 要点 |
|---|------|------|
| 1 | **弹性而非刚性** | 支撑/阻力线像橡皮筋，不是坚硬的墙 |
| 2 | **因果定律** | 震荡越久、区间越宽 → 突破后的趋势越剧烈 |
| 3 | **实时判断** | 用孤立支点识别震荡开始，不需要等"事后" |

## 七、房屋比喻 — 支撑/阻力的角色转换

**核心规律：突破后角色互换**
- 阻力位被突破 → 变成支撑位
- 支撑位被跌破 → 变成阻力位

```
突破前：天花板（阻力位）──突破──→ 天花板变成地板（支撑位）
```

## 八、为什么支撑/阻力如此重要

1. **突破交易** — 放量突破是绝佳的入场时机
2. **天然保护** — 突破后的旧阻力位变成新支撑位，提供天然止损位
3. **头寸管理** — 帮助决定何时进场、持有、出场
4. **历史记忆** — 过去的震荡区间永久保留在图表中
5. **多时间跨度** — 时间跨度越长，支撑/阻力越重要

## 九、实战要点

- 震荡阶段是最令人激动的阶段，耐心等待突破
- 突破后回踩是正常的，只要伴随低成交量
- 做空时同样适用：地板被突破后变成天花板
- 永远用**成交量确认**突破的真实性
- 止损线可设在价格震荡区间的下方一点
