# 第五章：量价分析的全局视角

> "错误是最好的老师。你从成功中可学不到什么。" —— 莫尼什·帕伯莱

## 一、五个核心概念

| 概念 | 定义 | 关键特征 |
|------|------|---------|
| **吸筹** | 局内人低价买入填满仓库 | 横盘震荡区间，反复"摇树"震出卖方 |
| **派筹** | 局内人高价卖出清空仓库 | 价格在窄幅区间内反复波动，吸引买方 |
| **供给测试** | 吸筹后测试卖压是否已被吸收 | 调低定价 → **低成交量 = 成功** |
| **抛售高峰** | 派筹尾声的"最后的狂欢" | 高成交量 + 长上影线 + 低实体 |
| **买入高峰** | 吸筹尾声的最后买入冲刺 | 高成交量 + 长下影线 + 低实体 |

### 吸筹阶段详解
- 利用利空消息引发恐慌抛售
- 局内人在低价大量买入（批发价）
- 反复"摇树"：多次下跌→买入→恢复，直到所有卖方被震出
- 可能持续数天到数月（取决于市场和时间跨度）

### 派筹阶段详解
- 利用利好消息吸引买方
- 局内人在高价大量卖出（零售价）
- 价格缓慢上涨（走楼梯），恢复投资者信心
- 用贪婪替代恐惧，直到买方在顶部涌入

### 供给测试（吸筹后）
- 调低定价回到之前有大量卖盘的价格区间
- 低成交量 → 卖压已清除 → 可以安全上涨
- 高成交量 → 仍有卖方 → 需继续震仓，再次测试

### 需求测试（派筹后）
- 调高定价回到之前有大量买盘的价格区间
- 低成交量 → 买压已满足 → 可以安全下跌
- 高成交量 → 仍有买方 → 需继续派筹，再次测试

## 二、完整市场周期

```
快速下跌 → 吸筹阶段 → 买入高峰 → 供给测试 → 缓慢上涨 → 加速上涨 → 派筹阶段 → 抛售高峰 → 需求测试 → 快速下跌 → 循环往复
```

### 关键节奏
- **上涨慢，下跌快**
  - 上涨：走楼梯（需要时间恢复信心，利润最大化）
  - 下跌：坐滑梯（仓库已空，快速填满）
- **恐惧 → 贪婪 → 恐惧** — 局内人只操纵两种情绪
- 蛇爬梯子游戏：沿梯子慢慢爬上去，顺着蛇一路滑下来

### 抛售高峰（顶部）K线特征
- **长上影线 + 低实体 + 高/极高成交量**
- K线实体颜色不重要
- 反复出现2~3次
- 局内人努力推高价格、大量出货，但卖压导致价格回落

### 买入高峰（底部）K线特征
- **长下影线 + 低实体 + 高/极高成交量**
- K线实体颜色不重要
- 反复出现
- 局内人快速打压价格，然后大量买入推高收盘

## 三、测试的两种结果

| 测试类型 | 目的 | 成功表现 | 失败表现 |
|----------|------|---------|---------|
| **供给测试**（吸筹后） | 确认卖压已清除 | 低量下跌 → 安全上涨 | 高量下跌 → 需继续震仓 |
| **需求测试**（派筹后） | 确认买压已满足 | 低量上涨 → 安全下跌 | 高量上涨 → 需继续派筹 |

> 测试是量价分析中**最强烈的信号之一** — 局内人发出了"即将突破"的明确信号。

## 四、俄罗斯套娃：多时间跨度的周期嵌套

这些周期存在于**所有时间跨度**中：

```
1分钟图中的周期（数小时） ⊂ 15分钟图中的周期（数天） ⊂ 日线图中的周期（数周） ⊂ 周线图中的周期（数月）
```

- 快时间跨度的周期会传导到慢时间跨度
- 传导越远（跨越越多时间跨度），趋势越持久（威科夫因果定律）
- 如同石头投入池塘的涟漪向外扩散

## 五、市场操纵的本质（重要澄清）

**市场操纵 ≠ 操纵价格**

> 市场操纵 = 利用媒体制造**恐惧和贪婪**，引发散户的买卖行为

- 局内人不是垄断联盟，只是能同时看到买卖双方
- 成交量是他们**无法掩盖的**
- 局内人的核心问题：大量买入会使价格上涨（对自己不利），大量卖出会使价格下跌 → 必须分阶段操作

## 六、暗池交易

大型机构通过暗池掩盖大宗交易，但并不影响量价分析：
- 大量卖出若一次性执行会使价格暴跌（对自己不利）
- 所以必须分批操作，最终还是在成交量上体现

## 七、实用时间跨度组合

| 交易风格 | 推荐组合 |
|----------|---------|
| 超短线 | 5分钟 + 15分钟 + 60分钟 |
| 波段交易 | 60分钟 + 240分钟 + 日线 |
| 趋势交易 | 4小时 + 日线 + 周线 |

## 八、关键结论

- 量价分析的核心就是跟随局内人：在他们买入时买入，在他们卖出时卖出
- 极高成交量配合特定K线形态 = 市场即将反转的明确信号
- 一切都建立在供求关系的简单逻辑之上
