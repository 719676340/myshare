# 第一章：交易之中无新事

> "华尔街里无新事，因为投机的历史如同山一样久远。" —— 杰西·利弗莫尔

## 一、核心思想

量价分析不是新发明，根基已存在 **100多年**。价格 + 成交量 = 揭示市场真相的"爆炸性组合"。

## 二、三位关键人物及其贡献

| 人物 | 身份 | 核心贡献 |
|------|------|----------|
| **查尔斯·道** (Charles Dow) | 技术分析之祖 | 道氏理论三阶段：吸筹 → 公众参与 → 派筹；**成交量确认价格趋势** |
| **理查德·威科夫** (Richard Wyckoff) | 量价分析之父 | 三大定律（见下文）；盘口解读方法 |
| **杰西·利弗莫尔** (Jesse Livermore) | 华尔街传奇交易者 | 从行情纸带中发现价格与买卖行为的关系，1929年崩盘中赚得1亿美元 |

## 三、道氏理论三阶段

1. **吸筹阶段** — 所有更强趋势的起点，"聪明资金"低位买入
2. **公众参与阶段** — 趋势跟踪阶段，通常是三阶段中最长的
3. **派筹阶段** — 投资者蜂拥而入，"聪明资金"反向卖出

> 查尔斯·道基本原则：如果一只股票的价格波动伴随很低成交量 → 原因可能多样；如果伴随高成交量或逐渐增大的成交量 → 这是一种**有效变动**。

## 四、威科夫三大定律（本章重点）

### 1. 供求定律
- 需求 > 供给 → 价格涨
- 供给 > 需求 → 价格跌
- 类比：冬季大促销，价格下降，消费者一拥而上，消化超额供给

### 2. 因果定律
- 小因小果，大因大果
- 成交量变化的规模决定价格波动的幅度
- 类比：大浪拍船剧烈摇晃，小浪影响甚微

### 3. 投入产出定律
- 成交量（投入）与价格变动（产出）应保持和谐统一
- **如果不统一 → 异常 → 深究原因**
- 相当于牛顿第三定律：任何行为都有力度相等方向相反的作用力

## 五、第四位重要人物：理查德·奈伊 (Richard Ney)

- 曝光做市商（"局内人"）的操纵行为
- 著作：《华尔街丛林》、《华尔街帮派》、《造市者造市》
- 核心观点：做市商 = 批发商，低价批发买入（吸筹），高价零售卖出（派筹）

### 奈伊八条法则（精简版）
1. 专业人士期望以零售价格售出以批发价格购入的股票
2. 干得越久，积累越多资本进行批发买入、零售卖出
3. 传播媒介扩张 → 更多散户入场 → 波动性增强
4. 做市商通过媒体寻找新途径提升卖出技巧
5. 必须使股价进一步下跌来洗出投资者，获得足够筹码
6. 股价上涨要有足够吸引力来吸引公众注意，方便出货
7. 最活跃的股票需要更长时间的派筹
8. 经济发展将受制于越来越严重的金融危机

## 六、关键洞察

- **成交量是唯一不能被隐藏的指标** — 价格可以被操纵，但成交量不会撒谎
- 量价分析**适用于所有市场**：股票、外汇、期货、大宗商品
- 适用于**所有时间框架**：日内、波段、长期投资
- 过去的行情纸带 = 现在的电子盘口，本质信息一样
