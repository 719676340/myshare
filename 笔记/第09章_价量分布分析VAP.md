# 第九章：价量分布分析（VAP）

> "无论熊市还是牛市，永远要和市场站在一边。" —— 查尔斯·道

## 一、VAP 与 VPA 的区别

| | VPA（量价分析） | VAP（价量分布） |
|---|---|---|
| **视角** | 每根K线形成后的量价关系 | 每根K线内部各价位的成交量分布 |
| **横轴** | 时间 | 价格 |
| **纵轴** | 价格 | 成交量 |
| **形态** | 底部的成交量柱状图 | 左侧的横向成交量柱状图 |

### 核心原理
把一根成交量柱体"解剖"，看成交量究竟集中在哪个价位。

| 成交量集中位置 | 含义 |
|--------------|------|
| **集中在低位** | 更可能是买入成交量 → 支撑区域 |
| **集中在高位** | 更可能是卖出成交量 → 阻力区域 |
| **集中在中间** | 该价位是关键的价格震荡区域 |

## 二、VAP 的颜色含义

- **红色占比多** = 上涨K线数量多 → 买方主导
- **蓝色占比多** = 下跌K线数量多 → 卖方主导

## 三、VAP 的三大用途

### 1. 可视化支撑和阻力
- 之前支撑/阻力是"无形的屏障"
- VAP让这些屏障显形 — 成交量越大的区域 = 越重要的支撑/阻力

### 2. 判断价格震荡区域的强弱

| 价量分布特征 | 判断 |
|------------|------|
| 高成交量 + 窄价格区间 + 持续时间长 | **极强**的支撑/阻力 |
| 低成交量 + 宽价格区间 + 持续时间短 | **较弱**的支撑/阻力 |
| 持续时间短但成交量极高 | **异常** — 市场变弱的信号 |

### 3. 辅助风险管理
- 支撑强 → 可以放心入场，止损设在下方
- 支撑弱 → 暂时撤退，等待其他信号
- 帮助确定止损线的合理位置

## 四、实战判断要点

### 横向对比成交量
- 以最底部的震荡区域为基准
- 对比其他区域的成交量大小
- 成交量相近但持续时间短 = 异常信号

### 时间因素
- 持续时间长 + 成交量大 = 极强区域
- 持续时间短 + 成交量大 = 注意！可能有大量抛售/买入
- 持续时间短 + 成交量小 = 弱区域，容易突破

## 五、VAP 的定位

> VAP是VPA的**补充工具**，永远不能**替代**VPA。

使用流程：
1. 用VAP识别关键价格区域（支撑/阻力）
2. 用VPA确认价格行为（量价确认/异常）
3. 两者结合 = 完整的三维分析
