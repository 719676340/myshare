# 第六章：K线图与量价分析

> "我从不与纸带争辩。" —— 杰西·利弗莫尔

## 一、五条基本原则

| # | 原则 | 要点 |
|---|------|------|
| 1 | **影线优先** | 影线长度是首要关注因素，立刻显示强弱与情绪 |
| 2 | **无影线 = 强情绪** | 朝收盘价方向的市场情绪强势 |
| 3 | **实体大小 = 情绪强度** | 低实体 = 弱，高实体 = 强 |
| 4 | **位置决定含义** | 同一K线在不同趋势位置意义完全不同 |
| 5 | **量价验证** | 先看K线，再看成交量是确认还是异常 |

## 二、三种核心K线形态

### 1. 射击十字星（Shooting Star）— 看跌信号

```
    │  ← 上影线长
  ──┼── ← 低实体（开盘≈收盘）
    │
```

- **含义**：价格先涨后跌回开盘价附近 = 卖方战胜买方 = **弱势**
- **不会出现异常** — 永远只确认弱势，区别在于程度

| 成交量 | 含义 |
|--------|------|
| 低量 | 轻微弱势，可能只是短暂停滞 |
| 平均量 | 中等弱势，可能回调 |
| **高/极高量** | **局内人正在卖出！大动作即将发生** |

- **重复出现 = 指数级增强**：1个=注意，2个=警惕，3个=行动
- 也出现在**下跌趋势的小反弹中** = 局内人在抛售吸筹来的存货

### 2. 锤头线（Hammer）— 看涨信号

```
    │
  ──┼── ← 低实体（开盘≈收盘）
    │  ← 下影线长
```

- **含义**：价格先跌后涨回开盘价附近 = 买方战胜卖方 = **强势**
- **同样不会出现异常**

| 成交量 | 含义 |
|--------|------|
| 低量 | 轻微强势，反转幅度小 |
| 平均量 | 中等强势，可能有日内机会 |
| **高/极高量** | **局内人正在大力买入！买入高峰信号** |

- **关键提醒**：单根锤头线不一定终止下跌，市场如油轮需要时间停下

### 3. 长腿十字线（Long-Legged Doji）— 不明确/犹豫

```
    │  ← 长上影线
  ──┼── ← 十字（开盘=收盘）
    │  ← 长下影线
```

- **含义**：市场剧烈双向波动后收于开盘价 = **犹豫**
- **可能出现异常**（与射击十字星和锤头线不同！）

| 成交量 | 含义 |
|--------|------|
| **低量（异常！）** | 价格大幅波动但无投入 = **局内人在操控洗盘，未真实参与** |
| 平均量 | 正常犹豫，可能是反转信号 |
| 高量 | 真实犹豫，反转概率增大 |

- 非农数据等重大新闻发布时最常见
- **新闻验证法**：新闻后看成交量 — 有量=局内人参与可跟随；无量=异常，保持观望

## 三、其他关键K线形态

| 形态 | 含义 | 成交量验证 | 位置 |
|------|------|-----------|------|
| **高实体K线** | 市场情绪强烈 | 高量=确认；低量=警示（散户在追） | 任何位置 |
| **低实体K线** | 市场情绪弱势 | 低量=正常；高量=异常（局内人在发力） | 任何位置 |
| **吊人线** | 上涨趋势中的潜在弱势 | 需高量确认；后续跟射击十字星则验证 | **只在上涨顶部** |
| **放量止跌信号** | 下跌即将停止 | 连续数根K线量递增，局内人在"踩刹车" | 下跌末期 |
| **放量止涨信号** | 上涨即将停止 | 高/极高量，K线实体渐低，形成"弧线" | 上涨末期 |

### 吊人线 vs 锤头线（关键区别！）

同一形态，位置不同，含义完全相反：
- **锤头线** = 下跌底部 → 看涨信号
- **吊人线** = 上涨顶部 → 看跌信号（第四原则的完美体现）

### 放量止跌信号的K线序列

```
长阴线（量高）→ 长下影线K线 → 低实体+长下影 → 锤头线出现
              局内人开始"踩刹车"     →        刹车生效
```

### 放量止涨信号的K线序列

```
长阳线 → 实体渐低 → 弧线模式 → 射击十字星出现
         局内人持续卖出，越来越难维持动能
```

## 四、量价分析的三大功能

1. **进场** — 发现低风险入场点
2. **持仓** — 穿越回调，坚定持有（下跌中低量锤头线 = 不是反转，继续持有空头）
3. **出场** — 判断反转信号，及时离场

## 五、新闻与成交量的关系

- 新闻发布后看成交量是学习量价分析最快的方法之一
- 成交量激增 + 价格验证 = 局内人参与了，可跟随
- 价格变动但成交量不验证 = 异常，局内人在操控，保持观望
- 成交量永远与新闻相互印证
