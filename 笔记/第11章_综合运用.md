# 第十一章：综合运用

> "市场不依赖侥幸或运气。像战场一样，它靠的是概率和可能性。" —— 大卫·德雷曼

## 一、四种关键价格形态

### 1. 下降三角形 — 看跌信号
- 高点不断降低 = 弱势
- 水平支撑线在底部
- 大概率向下突破

### 2. 上升三角形 — 看涨信号
- 低点不断抬高 = 牛市情绪
- 水平阻力线在顶部
- 大概率向上突破
- 突破后阻力变支撑，提供天然止损位

### 3. 三角旗形 — 方向不确定
- 高点降低 + 低点升高 = 价格区间收窄
- 因果定律：收缩越久，突破越猛烈（"螺旋弹簧"）
- 无法预判方向，需等待确认
- 适合期权交易者（无方向策略）

### 4. 三重顶 / 三重底 — 反转信号

**三重顶**：三次测试同一阻力位失败
- 交易机会1：在阻力位卖空
- 交易机会2：如果突破 → 极强支撑位

**三重底**：三次测试同一支撑位后反弹
- 交易机会1：在支撑位买入
- 交易机会2：如果跌破 → 强阻力位

### 价格形态的共同特点
- 都提供两种交易机会（盘整中 + 突破时）
- 都通过量价分析确认
- 都适用于任何市场和任何时间跨度

## 二、支点的灵活性

> 支点是GPS导航，量价分析是地图和罗盘。

- 支点不一定完美出现
- **成交量才是判断盘整开始的真正工具**
- 支点只是辅助标识上下界
- 支点缺失时，用眼睛和常识画线
- 支撑/阻力线像橡皮筋，不是钢条

## 三、多时间跨度分析（核心实战技巧）

### 三图组合

```
        ┌──────────────┐
        │  30分钟图     │  ← 望远镜（宏观/基准/主导趋势）
        └──────────────┘
  ┌──────────┐ ┌──────────┐
  │ 15分钟图  │ │ 5分钟图   │
  │ 主交易图  │ │ 细节图    │
  └──────────┘ └──────────┘
```

### 三个层次的用途

| 时间跨度 | 作用 |
|----------|------|
| **慢图** | 判断主导趋势方向，发现信号更直观 |
| **中图** | 主交易决策图，连接宏观和微观 |
| **快图** | 观察价格运动的具体细节 |

### 涟漪效应

```
5分钟图发现变化 → 传导到15分钟图 → 传导到30分钟图
（先反映）        （主交易图确认）    （宏观确认）
```

- 传导越远 = 趋势越持久
- 顺大势 = 低风险；逆大势 = 高风险

### 实战要点

- 30分钟图1根K线 = 15分钟图2根 = 5分钟图6根
- 更慢的时间图更直观地发现信号
- 即使不用量价分析，多时间跨度本身也很强大

## 四、止损线的设置

- 突破后：止损设在盘整区间的相反一侧
- 三重顶卖空：止损设在阻力位上方
- 三重底买入：止损设在支撑位下方
- 自然形成的价格位置 = 最佳止损位

## 五、核心信息

> "如果你知道了量价分析的力量，同时将其与价格震荡的知识结合起来，那你就已经掌握了90%的内容。剩下的部分就需要你的练习。"
