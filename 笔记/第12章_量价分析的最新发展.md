# 第十二章：量价分析的最新发展

> "我最大的胜利不是赚到的金钱，而是一些无形的东西：我的判断是正确的，我成功预测到了未来，而且我遵守着一个明确的规划。" —— 杰西·利弗莫尔

## 一、三种新发展

### 1. 等量图（Equivolume Chart）— 理查德·阿姆斯，1994年

- X轴 = 成交量（而非时间）
- 成交量融入图形本身，不再是底部独立指标
- 盒子形状反映量价关系

| 盒子形状 | 含义 | 对应传统K线 |
|----------|------|------------|
| 窄+高 | 异常 — 大产出小投入 | 高实体 + 低成交量 |
| 宽+矮 | 异常 — 小产出大投入 | 低实体 + 高成交量 |
| 正常比例 | 确认 — 量价协调 | 正常量价关系 |
| 宽+高 | 强力确认 | 高实体 + 高成交量 |

**作者评价**：
- 优点：直观展示量价关系
- 缺点：去掉了时间因素，丢失了因果定律的核心要素

### 2. 量烛图（Volume Candle）— 等量图 + K线图的结合

- 保留K线图的开盘、最高、最低、收盘价和影线
- 宽度反映成交量大小
- 作者认为比纯等量图更好

### 3. 差量法与累积差量图

- **差量** = 卖盘成交量 - 买盘成交量（净差值）
- **累积差量** = 所有差量累加，展示买卖力量演变
- 最适合有公开买卖盘信息的市场（期货、股票）
- 目前多为付费工具

## 二、全书最终要点

| 核心理念 | 要点 |
|----------|------|
| 量价分析是艺术 | 不能用软件替代 |
| 只找两样东西 | 确认 vs 异常 |
| 成交量不可隐藏 | 唯一的"真相指标" |
| 适用于一切 | 任何市场、任何时间跨度 |
| 需要练习 | 一旦学会终生不忘，但无捷径 |

> "成交量和价格共同构造出来的指标是揭示市场情绪以及主力活动的唯一指示器。没有量价分析的帮助，你在市场中就如盲人一样。"
